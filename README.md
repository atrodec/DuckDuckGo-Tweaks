# DuckDuckGo Tweaks
DuckDuckGo Tweaks is a customizable userstyle that improves DuckDuckGo. It removes visual clutter and other distractions. 🦆

## [Install with Stylus via UserStyles.world](https://userstyles.world/style/17992/duckduckgo-tweaks)

![Cover image](https://userstyles.world/preview/17992/4.webp)

## Features
### Removes:
- Search result ads
- Product promotions
- Feedback prompts
- Tips
- Footer
- Related searches (optional)
- Image dimensions (optional, visible by default)
- Other visual distractions

### Logos:
Offers the ability to remove the DuckDuckGo logo or set a custom logo.

Use CSS syntax in the "Custom logo" field. For example `url(https://example.com/logo.svg)`
